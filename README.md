**[product-os](https://github.com/namankumar/product-os)** — An AI harness for product work. A system of 21 skills to surface product ideas, sharpen positioning, and drive the dev cycle from 0 to 1. Each skill is a markdown file — plain instructions Claude Code reads and executes. Fork it, and a small team runs like a much larger one.

**[alloy](https://github.com/namankumar/alloy)** — What Claude Code is for coding, Alloy is for knowledge work. An agent harness with a GUI and web layer — skills, context, orchestration, file-based output — built for product teams instead of engineers. Multiplayer at the foundation. Self-hostable. Own your AI workflow, not rent it. Spec phase.

**[private-defi-agent](https://github.com/namankumar/private-defi-agent)** — Privacy-preserving agent for cross-chain fills: yield, prediction markets, perps. Built on state machines — a policy engine computes legal actions at each phase, Claude reads on-chain state and picks one. The model decides; the policy enforces.

**[privacy-score](https://github.com/namankumar/privacy-score)** — Every blockchain analytics tool tells you what it can learn about a wallet. This answers the inverse. Paste an Ethereum address, get a 0–100 privacy score across six categories — entity exposure, identity leaks, timing patterns, cross-chain linkability — with a plain-English breakdown of what's exposed and how to fix it.

**[bloom-tutor](https://github.com/namankumar/bloom-tutor)** — Benjamin Bloom's 1984 study found 1-on-1 tutoring puts students two standard deviations ahead of classroom peers. Almost no child gets it. A daily AI tutor on Claude voice mode — persistent session log, five-day curriculum, math game. 30 minutes a day.
