**[privacy-score](https://github.com/namankumar/privacy-score)** — Every blockchain analytics tool tells you what it can learn about a wallet. This answers the inverse. Paste an Ethereum address, get a 0–100 privacy score across six categories: entity exposure, identity leaks, timing patterns, cross-chain linkability. Plain-English breakdown of what's exposed and how to fix it.
